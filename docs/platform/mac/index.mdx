## 安装

双击 pkg 安装包并一路点按「继续」，之后输入你的密码或使用指纹允许安装；
等待安装过程结束，你就可以在「应用程序」文件夹或 Launchpad 中找到 Koishi 了。

## 启动

点按 Koishi 图标即可启动。等待几秒钟，Koishi 将启动并弹出控制台窗口，
现在你可以开始使用 Koishi 了。
