## 安装

点按你下载的安装包，「应用安装程序」会启动并进行安装。

安装完成后，点按启动器内的 Koishi 图标，并等待 Koishi 完成初始化。
此过程可能需要十分钟或更久，在此期间你需要保持设备常亮，
且 Koishi 始终处于前台。

## 启动

点按 Koishi 图标，然后在管理页面内选择「启动」即可启动 Koishi。
Koishi 启动后，退出管理页面并选择进入 Koishi 控制台，
现在你可以开始使用 Koishi 了。
