Koishi Desktop 不支持 Windows 7。

但是，有 Koishi 社区的贡献者制作了 Koishi Desktop 的 Win7 修改版本，你可以在这里下载。

:::danger

此修改版本不属于 Koishi.js 的官方项目，因此无法获得 Koishi.js 的支持。这意味着：

- 你将不会获得来自 Koishi Desktop 的更新，包括安全更新。
- 如果有插件或其他功能出现了问题，你将不会获得它们的升级或修复。

:::
