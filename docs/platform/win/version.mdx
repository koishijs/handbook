下面列出了 Koishi 桌面所支持的 Windows 版本及该版本所支持的 Koishi 桌面功能。

同一产品只列出了其最高版本。

## 图例

| 标识               | 含义                             |
| ------------------ | -------------------------------- |
| :white_check_mark: | 功能由官方支持。                 |
| :warning:          | 功能有额外的运行条件或注意事项。 |
| :x:                | 功能不支持或不由官方支持。       |
