Koishi 社区的贡献者制作了 Koishi Desktop 的一键安装脚本，该脚本对解决某些问题很有帮助。

:::danger

此脚本不属于 Koishi.js 的官方项目，因此无法获得 Koishi.js 的支持。这意味着：

- 你将不会获得来自 Koishi Desktop 的更新，包括安全更新。
- 如果有插件或其他功能出现了问题，你将不会获得它们的升级或修复。

:::
