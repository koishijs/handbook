QQ 频道有两种接入方法：

1. 使用 `onebot` 插件：使用自己的 QQ 号登录机器人，无需向腾讯申请，有风控风险
2. 使用 `qqguild` 插件：需要 [向腾讯申请](https://koishi.js.org/plugins/adapter/qqguild.html#接入方法)，使用腾讯提供的机器人登录方式，没有风控风险，但各项功能均受到限制（如不能主动发送消息）
