QQ频道有两种接入方法哦
1. 使用onebot插件：无需向腾讯申请，使用QQ号登录，有风控风险
2. 使用qqguild插件：需要 [向腾讯申请](https://koishi.js.org/plugins/adapter/qqguild.html#接入方法) ，使用腾讯提供的机器人登录方式，没有风控风险，只能发受限的消息，消息不能想发就发
