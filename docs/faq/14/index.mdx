配置文件所在的目录叫根目录。根据你的安装方式，根目录的位置可能不同：

- zip 便携包：解压目录就是你的根目录
- msi 安装包：`C:/Users/你的用户名/AppData/Roaming/Il Harper/Koishi`
- pkg 安装包：`~/Library/Application Support/Il Harper/Koishi`

配置文件是根目录下名为 `koi.yml` 的文件。当你遇到问题时，开发者可能会要求你提供配置文件的内容。此时去上面的地方找就好了。
