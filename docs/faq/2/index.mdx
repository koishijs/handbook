如果你在 Windows 上遇到 Koishi
直接显示「安装失败」的情况，那么这可能是由于你的电脑环境故障所致。你需要参考『[安装插件时直接显示「安装失败」](/faq/19)』进行解决。

如果插件安装了若干分钟，然后显示「安装失败」，那么实际可能已经安装成功，重启
Koishi 即可在「插件配置」中添加刚刚安装的插件了。

![安装失败](installation-failed.png)

如果重启后在「插件配置」的「添加插件」下拉框中仍无法找到插件，
尝试查看控制台中的「日志」视图中所输出的日志。里面可能输出了插件加载失败的原因。
