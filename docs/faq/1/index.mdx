欢迎使用 Koishi 常见问题解答（FAQ）！
这里列举了一些大家在使用 Koishi 时经常遇到的问题，并对其做出了详细的解答。

你可以使用浏览器的 **页面内搜索** 功能搜索 FAQ 主题，
或直接进行 [全站搜索](/search)。

## 请先确保……

- 你使用的是 Koishi Desktop 或 Koishi Android 的最新版本。Koishi Desktop 或 Koishi Android 的最新版本可以在本站的 [首页](/) 下载，下载时请注意下载 Koishi Desktop 的「安装版」或 Koishi Android 的「标准版」。

- 你的 Koishi、控制台和所使用的插件均已更新到了最新版本。开发者可能已经在最新版本中修复了你遇到的问题。

- 已经尝试过移除并重新添加插件，或重启 Koishi。

- 你已经详细阅读过控制台中的「日志」视图中所输出的日志。里面很有可能已经包含了解决你遇到的问题的提示。

- 如果无法访问或登录控制台，可以尝试「[Koishi 在重启之后没有弹出控制台页面](/faq/8)」和「[无法登录控制台](/faq/9)」。

最后，如果 FAQ 中没有你遇到的问题，你可以 [向社区提问](/community)。
