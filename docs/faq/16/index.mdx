有时你可能搞乱了配置或者一些文件，但是又不知道搞坏了哪里，此时一次重新安装可能会解决这些问题。

你可以按照下面的方法进行全新安装：

- zip 便携包：
  1. 停止你的 Koishi-desktop 程序
  2. 删除你解压出来的整个文件夹
  3. 下载最新的 Koishi-desktop [下载链接](https://koishi.chat/manual/starter/)
  4. 按照指引进行安装
- msi 安装包：
  1. 停止你的 Koishi-desktop 程序
  2. 卸载 Koishi-desktop 程序
  3. 删除 `C:/Users/你的用户名/AppData/Roaming/Il Harper` 文件夹
  4. 下载最新的 Koishi-desktop [下载链接](https://koishi.chat/manual/starter/)
  5. 按照指引进行安装
- pkg 安装包：
  1. 停止你的 Koishi-desktop 程序
  2. 卸载 Koishi-desktop 程序
  3. 删除 `~/Library/Application Support/Il Harper` 文件夹
  4. 下载最新的 Koishi-desktop [下载链接](https://koishi.chat/manual/starter/)
  5. 按照指引进行安装
